## 📦 Despliegue en GitHub Pages

1. Sube todos los archivos a un repositorio de GitHub.
2. Ve a **Settings → Pages**.
3. En "Branch", selecciona `main` y la carpeta `/ (root)`.
4. Haz clic en **Save**.
5. Tu sitio estará disponible en `https://<tu-usuario>.github.io/<nombre-repo>/`

## ✏️ Personalización

- **Número de WhatsApp**: Busca `wa.me/1234567890` en `index.html` y reemplaza con tu número real (formato internacional, sin espacios ni `+`).
- **Logo**: Reemplaza `assets/logo.svg` con la imagen oficial del logo.
- **Colores**: Todas las variables de color están en `styles.css` bajo `:root`.
